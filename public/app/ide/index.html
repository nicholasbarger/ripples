Information

Ripple Builder let's you compose independent chunks of code into short and long running processes. Each code is ran in it's own container, so each segment can use a different language or toolset that best suits it's needs. Ripples can be composed of code you provide or easily pulled down from the gallery.

All changes are version-controlled and easily forked to allow customizations.
